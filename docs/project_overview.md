# **End-to-End Clinical Data Pipeline (EDC → SDTM → ADaM → TLF)**



#### Objective



Build a reproducible pipeline in SAS to transform raw clinical data into SDTM and ADaM datasets and generate basic TLF outputs.



#### **Study Context**



Therapeutic Area: Diabetes

Design: Randomized, 2-arm (Drug vs Placebo)

Duration: 12 weeks

Primary Endpoint: Change in glucose levels (VS)

Safety: Adverse Events (AE)
